## 今日技術決策

### Supabase key

本專案使用新版 Supabase API Keys，不再使用 legacy anon / service role 變數名稱。

使用原則：

* Browser client 使用 publishable key。
* 一般 server session client 使用 publishable key。
* Admin API route 才使用 secret key。
* Secret key 不得出現在 client component。

### 問卷結果頁語氣

結果頁遵守 `strategy/quiz_diagnosis_logic.md`：

* 專業診斷。
* 溫和警示。
* 可執行建議。
* 不使用單純恐嚇或精準延畢機率。

### AI 指令產生器 Phase 1B

AI 指令產生器維持既定決策：

* 不在 RAPID 上傳 PDF。
* 不呼叫後端 LLM API。
* 使用純前端字串模板拼接。
* 使用者自行把生成的指令與 PDF 上傳到外部 AI。
* 目前已支援 ChatGPT、Claude、Gemini、Grok 四種外部 AI 選擇。
* 目前已支援教授追問、邏輯稽核、簡報修改、英文潤飾四種指令方向。
